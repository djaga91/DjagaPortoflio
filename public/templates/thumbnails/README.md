# Miniatures des Templates

Ce dossier contient les miniatures de prévisualisation pour les templates de portfolio.

## Structure recommandée

Placez vos images avec les noms suivants :
- `template1.jpg` ou `template1.png` - Miniature pour le Template Moderne
- `template2.jpg` ou `template2.png` - Miniature pour le Template Minimaliste
- `template3.jpg` ou `template3.png` - Miniature pour le Template Créatif
- `template4.jpg` ou `template4.png` - Miniature pour le Template Artistique
- `template5.jpg` ou `template5.png` - Miniature pour le Template Système
- `template6.svg` - Miniature pour le Template Élégant (marron/beige, incluse par défaut)

## Format recommandé

- **Taille** : 1200x900px (ratio 4:3)
- **Format** : JPG ou PNG
- **Poids** : < 500KB (optimisé pour le web)

## Utilisation

Les miniatures sont automatiquement chargées depuis ce dossier. Si une miniature n'existe pas, le système utilisera l'URL Unsplash par défaut comme fallback.

## Ajout d'une nouvelle miniature

1. Exportez votre screenshot du template au format JPG ou PNG
2. Nommez le fichier selon le format : `templateX.jpg` (où X est le numéro du template)
3. Placez-le dans ce dossier
4. Le système l'utilisera automatiquement
