🔐 Erreur de connexion SSL

Vous voyez probablement un message d'erreur comme celui-ci :

« Votre connexion n'est pas privée »

Code d'erreur : NET::ERR_CERT_AUTHORITY_INVALID

🤔 Pourquoi cette erreur ?

Cette erreur apparaît généralement quand vous êtes connecté à un réseau d'entreprise, d'école ou d'université qui utilise un proxy d'inspection SSL.

Ce n'est PAS un problème avec PortfoliA. Notre certificat SSL est parfaitement valide et noté A+ par SSL Labs (la meilleure note possible). C'est le réseau sur lequel vous êtes qui intercepte votre connexion.

🏆 Notre certificat SSL est vérifié A+

  • ✅ Émetteur : Let's Encrypt (autorité mondiale reconnue)
  • ✅ Protocole : TLS 1.3 (le plus sécurisé)
  • ✅ HSTS activé (protection maximale)
  • ✅ Aucune vulnérabilité connue

Vérifiez par vous-même : SSL Labs Test

Comment ça fonctionne ?

Connexion normale (chez vous) : ┌─────────────┐ HTTPS sécurisé ┌─────────────┐ │ Votre PC │ ────────────────────────► │ portfolia.fr│ └─────────────┘ Certificat Let's Encrypt └─────────────┘ ✅ Valide Connexion via proxy (école/entreprise) : ┌─────────────┐ Certificat ┌─────────────┐ Certificat ┌─────────────┐ │ Votre PC │ ───────────────► │ Proxy école │ ───────────────► │ portfolia.fr│ └─────────────┘ de l'école └─────────────┘ Let's Encrypt └─────────────┘ ❌ Non reconnu ↑ Le proxy "ouvre" votre connexion

✅ Solutions

📱 Solution 1 : Utilisez vos données mobiles RECOMMANDÉ

La solution la plus simple et la plus rapide :

  • Désactivez le Wi-Fi de votre appareil
  • Activez les données mobiles (4G/5G)
  • Ou créez un partage de connexion depuis votre téléphone

🏠 Solution 2 : Connectez-vous depuis un autre réseau

Essayez d'accéder à PortfoliA depuis :

  • Votre domicile
  • Un café ou lieu public avec Wi-Fi
  • Un autre réseau non filtré

🔧 Solution 3 : Demandez à votre service informatique

Si vous devez absolument utiliser le réseau de votre école/entreprise :

  • Contactez le service informatique
  • Demandez l'ajout de portfolia.fr à la liste blanche SSL
  • Ou demandez le certificat racine à installer sur votre appareil
⚠️ Attention : N'installez un certificat racine que si vous faites confiance à l'organisation. Cela leur permet de voir tout votre trafic HTTPS.

❓ Questions fréquentes

Est-ce que mes données sont en danger ?

Non. Chrome a bloqué la connexion avant tout échange de données. Vos mots de passe et informations n'ont pas été envoyés.

Est-ce que PortfoliA est piraté ?

Non. Notre site utilise un certificat SSL valide émis par Let's Encrypt. C'est le proxy de votre réseau qui cause ce problème.

Pourquoi l'école fait ça ?

Les établissements utilisent l'inspection SSL pour :

  • Filtrer les contenus inappropriés
  • Détecter les menaces de sécurité
  • Surveiller l'utilisation du réseau

C'est une pratique courante et légale dans le cadre éducatif ou professionnel.
