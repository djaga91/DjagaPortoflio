# Images Preview CV

Pour que les aperçus des templates CV fonctionnent, placez les images dans ce dossier :

## Images requises

1. **`cv-preview-modern.jpg`** - Aperçu du template Modern (2 colonnes)
2. **`cv-preview-latex.jpg`** - Aperçu du template Academic LaTeX (1 colonne)

## Instructions

1. Générez un CV avec chaque template depuis votre compte `john.doe@efrei.net`
2. Faites une capture d'écran ou exportez le CV en image
3. Placez les images dans `frontend-game/public/` avec les noms ci-dessus
4. Les aperçus s'afficheront automatiquement dans la page de sélection de templates

## Format recommandé

- Format : JPG ou PNG
- Taille : Recommandé 1200x1600px (ratio A4)
- Qualité : Haute qualité pour un rendu net
