# 🎨 Guide d'utilisation du Générateur de Portfolio

## 📍 Comment accéder à votre Portfolio

1. **Dans la barre latérale gauche** (ou menu mobile en bas), cliquez sur **"Mon Portfolio"** 📁
2. Vous verrez votre portfolio généré automatiquement

## 🎯 Modes disponibles

### 👁️ Mode Visualisation (par défaut)
- Affiche votre portfolio tel qu'il apparaîtra publiquement
- Toutes les sections avec vos données sont visibles

### ⚙️ Mode Édition
- Cliquez sur le bouton **"⚙️ Édition"** en haut à droite
- Une barre latérale gauche apparaît pour personnaliser

## 🖱️ Comment utiliser le Drag & Drop

### Option 1 : Glisser-Déposer
1. **Cliquez et maintenez** sur l'icône ☰ (trois barres verticales) à gauche de chaque section
2. **Glissez** la section vers le haut ou vers le bas
3. **Relâchez** pour placer la section à la nouvelle position

### Option 2 : Boutons flèches
1. **Survolez** une section avec la souris
2. Des flèches ↑ et ↓ apparaissent à droite
3. **Cliquez** sur ↑ pour monter ou ↓ pour descendre

### Option 3 : Masquer/Afficher
- Cliquez sur l'icône 👁️ à droite de chaque section
- Une section masquée ne s'affichera pas sur votre portfolio public

## 💾 Sauvegarde

- **Automatique** : Vos modifications sont sauvegardées automatiquement après 2 secondes
- **Manuelle** : Cliquez sur le bouton **"💾 Sauvegarder"** pour sauvegarder immédiatement

## 📋 Sections disponibles

- 👤 **Hero** : Section principale avec votre photo et nom
- 📝 **À propos** : Votre biographie
- 💼 **Expériences** : Vos expériences professionnelles
- 🎓 **Formation** : Votre parcours éducatif
- 🚀 **Projets** : Vos projets portfolio
- ⚡ **Compétences** : Vos compétences techniques
- 🌍 **Langues** : Langues que vous maîtrisez
- 🏆 **Certifications** : Certifications obtenues
- 📧 **Contact** : Liens sociaux (LinkedIn, GitHub, etc.)

## ❓ Questions fréquentes

**Q : Mon portfolio est vide ?**  
R : Assurez-vous d'avoir rempli votre profil (bio, expériences, projets, etc.) dans la section "Mon Profil"

**Q : Les sections ne bougent pas ?**  
R : Assurez-vous d'être en **Mode Édition** (bouton ⚙️ Édition en haut à droite)

**Q : Comment voir mon portfolio en public ?**  
R : Cette fonctionnalité sera disponible bientôt ! Pour l'instant, vous pouvez le voir en mode visualisation.
