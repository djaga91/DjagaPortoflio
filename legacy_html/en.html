My Projects

AirportProject (Scala)

Airport Management Application

Scala application that parses three CSV files (countries, airports, runways) and provides search by code or fuzzy name, as well as report generation.

  • Custom CSV Parsing (no external lib)
  • Fuzzy matching (partial name) via me.xdrop.fuzzywuzzy
  • Reporting capabilities (Top 10 countries with most/fewest airports, etc.)
  • JavaFX GUI and a CLI mode
  • Scala Futures for parallel data processing

Run with sbt clean compile run (GUI by default) or adjust mainClass for the CLI version. Includes test suites via sbt test.

GitHub Repository

Diabetes Machine Learning

Diabetes Classification with scikit-learn

A project carried out at EFREI to predict diabetes onset using the Pima Indians Diabetes dataset. I performed an exploratory analysis (EDA), data preprocessing (missing values, standardization), and implemented various classification models:

  • Logistic Regression
  • Decision Tree
  • Neural Network (MLP)

I then tuned hyperparameters (Grid Search) and compared accuracy, precision, recall, F1-score, and confusion matrices. The optimized Neural Network showed the best performance for detecting diabetic patients.
